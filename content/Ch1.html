Ch1

在 Windows 10 操作環境下, 使用者可以直接在操作系統安裝 Python 解譯器, 也可以使用可攜式 Python.

直接安裝 Python:

從 https://www.python.org/downloads/windows/ 下載最新版本後進行安裝. 可以選擇安裝包括 Documentation, pip (Package installer for Python) 以及其他所有內容. 安裝結束後, 開啟系統命令列, 輸入 python 即可進入互動模式, 要離開互動模式, 利用鍵盤 Ctlr + z 後再按 Enter 就可離開互動模式.

假如使用 Windows 7 操作系統, 則只能安裝 Python 3.8 版之前的套件.

使用可攜式 Python:

若使用 USB 3.1 以上的外接隨身碟, 則可以在將 Python 安裝至操作系統後, 直接複製到隨身碟中, 透過設定就可以隨身帶著 Python 解譯器. 只是 Python 的安裝流程會在 pip.exe 綁入操作系統 python.exe 的絕對執行路徑, 因此針對要複製到隨身碟, 以可攜式模式使用 Python 的操作系統安裝, 在系統安裝時不可以直接安裝 pip, 而必須在安裝其他模組後, 選擇以手動執行 python get-pip.py 的方式安裝 pip.

至於有關可攜程式系統的製作, 通常以批次檔案設定, 採用 subst 指令, 將可攜系統的資料目錄與特定磁碟代號對應, 可以在隨身碟插入 USB 後以 start.bat 啟動隨身系統, 避開在不同 Windows 10 操作系統時可能給定隨身碟不同代號的設定問題.

假如使用 Mac 或 Linux 操作系統時, 必須注意操作系統中已經同時安裝 Python 2.7 與 Python 3.8 等兩個版本的狀況, 執行 Python 3.8 時, 必須使用 python3, 而且安裝模組時, 必須使用 pip3 指令.

當使用者可以在命令列中執行 python 或 python3 進入互動模式 (Interactive Mode) 後, 就可以配合教科書中的 Ch1, 逐一練習 Python 指令, 將 Python 互動模式當作計算機使用.

解譯器與編譯器

解譯器: Interpreter - Python, Javascript

編譯器: Compiler - C/C++

Python 的縮排 (Identation):

C/C++ 採用 { } 界定程式迴圈或判斷式的執行範圍.

Python 則採用四個 spaces 為一個單位的縮排來界定迴圈或判斷式的執行範圍.
